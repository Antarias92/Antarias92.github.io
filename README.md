Project Overview
This project is a personal portfolio. My goal is to create a site that can showcase the projects I have worked on to potential employers. 
The website allows the user to navigate through four pages, scroll through a carousel, watch a video, and send emails through a contact form.
Technology Used: This project was created using HTML, CSS, Bootstrap, and Formspree (for contact validation).

User Stories
1.	As a prospective university student, I want to see what type of projects I will be working on during my undergraduate course, so that I can determine if Arts, Technology, and Emerging Communications is for me.
2.	As a job recruiter, I want to see what types of projects this candidate has completed, so that I may determine if their skill set is right for the job.
3.	As an instructor, I want to navigate a responsive website, so that I know the elements of that have been introduced 

Future Improvements
1.	I want to learn more UI principles to feature non-visual projects and have them appear presentable.
2.	Incorporate animations when a user clicks on an object or navigates the website.
3.	I would like to enhance the site with JavaScript.
